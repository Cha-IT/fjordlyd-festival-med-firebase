# Fjordlyd-festival med firebase

Ta utgangspunkt i koden i repoet (det er den samme som ble gjennomgått i timen). Lag en oversikt over hvilke artister som spiller på hvilke dager. Du legger inn hvilke artister du vil via Firebase. Husk å koble til din egen Firebase Firestore-database i *firebaseConfig.js*-filen

## Slik åpner du prosjektet på din datamaskin
1. Klon repoet i VS Code
2. Åpne terminal
3. Skriv `npm install` i terminalen for å installere avhengigheter (dvs ekstra programvare som hører til prosjektet og som er definert i *package-lock.json*)
4. Skriv `npm run dev` for å kjøre webserveren på _localhost_.
5. **Vite** vil automatisk oppdatere nettsiden når du lagrer endringer, og automatisk restarte serveren når det er nødvendig, så du kan bare la serveren kjøre mens du jobber.
